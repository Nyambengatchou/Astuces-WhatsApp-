Comment utiliser WhatsApp Web

Envoyez et recevez vos messages WhatsApp directement depuis votre ordinateur, sans application à installer.

💡 Prérequis : Votre téléphone doit rester connecté à Internet (Wi-Fi ou données mobiles) pour que WhatsApp Web fonctionne.

Se connecter à WhatsApp Web

  1. 1
    Ouvrez votre navigateur (Chrome, Firefox, Edge…) et rendez-vous sur web.whatsapp.com
  2. 2
    Un QR code s'affiche à l'écran. Gardez cette page ouverte.
  3. 3
    Sur votre téléphone, ouvrez WhatsApp → appuyez sur les 3 points (Android) ou Paramètres (iPhone)
  4. 4
    Appuyez sur Appareils connectés → Connecter un appareil
  5. 5
    Pointez votre téléphone vers le QR code sur l'écran. La connexion s'établit en quelques secondes !

Fonctionnalités de WhatsApp Web

💬

Messages & médias

Envoyez du texte, des photos, des vidéos, des documents et des vocaux depuis votre ordinateur.

🔔

Notifications bureau

Recevez des notifications Windows/Mac pour chaque nouveau message, même si l'onglet est en arrière-plan.

🔍

Recherche rapide

Utilisez Ctrl+F pour rechercher un message ou un contact dans toutes vos conversations.

🌙

Mode sombre

Activez le mode sombre : icône ⚙️ → Thème → Sombre. Plus reposant pour les yeux la nuit.

Se déconnecter de WhatsApp Web

Pour protéger votre vie privée, déconnectez-vous toujours après utilisation sur un ordinateur partagé.

  1. 1
    Dans WhatsApp Web : cliquez sur les 3 points verticaux → Se déconnecter
  2. 2
    Depuis le téléphone : WhatsApp → Appareils connectés → sélectionnez l'appareil → Se déconnecter

WhatsApp Web vs Application de bureau

🌐 WhatsApp Web

  • Aucune installation requise
  • Fonctionne dans tous les navigateurs
  • Idéal pour une utilisation ponctuelle
  • Accès depuis n'importe quel PC

💻 Application bureau

  • Disponible sur Windows & Mac
  • Notifications plus fiables
  • Meilleure intégration système
  • Idéal pour usage quotidien
⚠️ Sécurité : Sur un ordinateur partagé (cybercafé, bureau…), déconnectez-vous TOUJOURS et ne cochez jamais « Rester connecté ».
